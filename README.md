
# Riize API REST

API pour l'application mobile "Riize"


## Versions
- Node : 22.13.0
- npm : 10.9.2
- Express : 5.1.0

## Installation

Initialiser npm sur le projet et installer les dépendences 

```bash
  npm init -y
  npm ci
```

## Liste des dépendances  

```bash
  Express
  Cors
  BodyParser
  Mysql2
  Dotenv
  Swagger
  JsonWebToken
  BcryptJS
  GoogleAuthLibrary
  Formidable
  NodeMailer
  Axios
  Form-data
```

## Serveur

Lancer le serveur local  

```bash
  node server.js
```
